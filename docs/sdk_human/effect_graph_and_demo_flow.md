# Effect Graph and Demo Flow (Human SDK)

Aurex-X can now shape a scene like a live performance pipeline.

## Effect Graphs
Think of an effect graph as a procedural chain:
Generator → Modifiers → Patterns → Fields → Materials → Lighting.

This lets you create a scene's "personality" with deterministic building blocks.

## Automation
Automation tracks connect musical/time signals to parameters:
- bass can inflate tunnel radius
- phrase can raise lighting intensity
- beat can animate camera feel

## Demo Sequencer
The demo sequencer arranges multiple scene blocks and transitions so a full show can be authored from pure math.

Use it for arc-based storytelling: intro, buildup, drop, finale.
