# Music Controls Time (Human Guide)

Rhythm-Space lets the beat control not only visuals, but **time flow**.

## What it does
- Beat pulses can make geometry breathe.
- Measures and phrases can trigger larger structure changes.
- Strong beats can create short-lived echo duplicates.
- Time warp can speed up, delay, or reverse motion feel.

## Main controls
- `beat_geometry`: turn beat-driven geometry pulsing on/off.
- `echo_effect`: turn kick-like visual echoes on/off.
- `particle_mode`: choose rhythmic particle character (`Bass`, `Snare`, `Melody`).
- `time_warp`: tune scale/delay/echo/reverse of scene-time behavior.

## Quick starter recipe
1. Enable `beat_geometry` and `echo_effect`.
2. Set `particle_mode` to `Bass` for punchy pulses.
3. Add mild `time_warp` (`time_scale ~ 1.05`, `time_echo ~ 0.15`).
4. Increase gradually; keep it readable.
