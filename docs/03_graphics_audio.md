# Aurex Graphics and Audio Systems

GRAPHICS

Geometry is generated procedurally through the Shape Synth Unit.

Supported primitives include:

circle
rectangle
polygon
ring
extrusion
tube
grid
fractal structures

Materials are procedural styles:

flat
neon
chrome
crystal
noise
plasma
wireframe

Lighting system:

max lights: 12
types:
ambient
point
spot
pulse

Lighting interacts with geometry and particles to create stylized visuals.

Post Processing Effects:

bloom
motion trails
fog
color grading
screen distortion
CRT style filters

These effects allow simple geometry to produce rich visual scenes.

AUDIO

Audio is produced by the Aurex Sound Unit (ASU).

Capabilities:

synth voices
drum generators
pattern sequencer
procedural instruments

Audio events can trigger visual systems.

Example sync events:

beat
kick
snare
drop

Visual systems such as lighting, particles, and camera motion
can react to these events.

The Prime Pulse itself emits a rhythmic signal that subtly influences
environmental animation and audio behavior.
